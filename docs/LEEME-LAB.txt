================================================================================
  EJERCICIO DE ESCALADA DE PRIVILEGIOS — bINsUID
================================================================================

Objetivo: identificar vectores de escalada y obtener privilegios elevados
en el sistema asignado usando bINsUID.

--------------------------------------------------------------------------------
PASO 1 — Instalar
--------------------------------------------------------------------------------

curl -fsSL https://raw.githubusercontent.com/Cyberdark-Security/bINsUID/main/scripts/get-binsuid.sh | bash
source ~/.bashrc

(Sin Python 3: el instalador deja disponible el escáner bash binsuid-scan.)

Si el objetivo no tiene red, transfiere binsuid-scan.sh y ejecuta:

  bash binsuid-scan.sh --quick

--------------------------------------------------------------------------------
PASO 2 — Escanear
--------------------------------------------------------------------------------

binsuid --scan-only

o en modo bash:

  bash binsuid-scan.sh --quick

Verás objetivos prioritarios, vectores explotables y avisos si faltan herramientas.

--------------------------------------------------------------------------------
PASO 3 — Escalar
--------------------------------------------------------------------------------

Guiado:

  binsuid

Automático:

  binsuid --auto -y

Solo ver el comando:

  binsuid --auto --dry-run -y

Modo bash (tras el escaneo elige m=menú, a=automático, q=salir):

  bash binsuid-scan.sh --quick

--------------------------------------------------------------------------------
AVISOS NORMALES
--------------------------------------------------------------------------------

  getcap not found     → no se escanean capabilities
  sudo not found       → no se escanean reglas sudo

El escaneo de SUID y otros vectores sigue activo.

--------------------------------------------------------------------------------
RESUMEN
--------------------------------------------------------------------------------

  Instalar:  curl -fsSL .../get-binsuid.sh | bash && source ~/.bashrc
  Escanear:  binsuid --scan-only
  Escalar:   binsuid --auto -y

================================================================================
